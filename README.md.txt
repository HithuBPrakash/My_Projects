# To-Do List Application

## Overview
A simple, responsive to-do list application built with HTML, CSS, and JavaScript. The user can add, delete, complete, and filter tasks in the list.

## Features
- Add a task
- Mark a task as complete or incomplete
- Delete a task
- Filter tasks: show all, only completed, or only incomplete tasks
- Responsive design

## Bonus Features
- Edit a task (not implemented in this version)

## How to Run Locally

1. Clone the repository:
    ```
    git clone https://github.com/yourusername/todo-app.git
    ```

2. Navigate into the project folder:
    ```
    cd todo-app
    ```

3. Open the `index.html` file in your browser:
    - Open your file explorer and double-click the `index.html`, or
    - Run it on a local server (you can use VSCode Live Server extension).

4. You're good to go!
